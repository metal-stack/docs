Deploying metal-stack

We are bootstrapping our control-plane as well as our partitions using Ansible through CI.

In order to build up your deployment, you can make use of the Ansible roles that we are using by ourselves in order to deploy the metal-stack. You can find them in the repository called metal-roles.
